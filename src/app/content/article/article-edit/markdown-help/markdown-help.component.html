Markdown Result Shortcut
**text** Bold Ctrl/⌘ + B
*text* Emphasize Ctrl/⌘ + I
~~text~~ Strike-through
[title](http://) Link Ctrl/⌘ + K
`code` Inline Code Ctrl/⌘ + Alt + C
![alt](http://) Image Ctrl/⌘ + Shift + I
* item List Ctrl/⌘ + L
1. item Ordered List Ctrl/⌘ + Alt + L
> quote Blockquote Ctrl/⌘ + '
==Highlight== Highlight
# Heading H1
## Heading H2 Ctrl/⌘ + H
### Heading H3 Ctrl/⌘ + H (x2)
